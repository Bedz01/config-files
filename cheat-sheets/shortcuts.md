# Shortcuts

## Editor

| Shortcut              | Description            |
| --------------------- | ---------------------- |
| Alt + Shift + Up/Down | Move line up/down      |
| Ctrl + D              | Duplicate line         |
| Ctrl + Y              | Delete line            |

## Browser

| Shortcut              | Description            |
| --------------------- | ---------------------- |
| Ctrl + Shift + Del    | Delete browser data    |
| Ctrl + Shift + C      | Inspect element        |

## IntelliJ IDEA

| Shortcut              | Description            |
| --------------------- | ---------------------- |
| Alt + H               | Show file history      |
| Alt + P               | Plugins                |
| Alt + V               | Split vertically       |
