# window.location

- `https://www.darekkay.com/projects/?filter=JS#2`

```
window.location.origin   → 'https://www.darekkay.com'
               .protocol → 'https:'
               .host     → 'www.darekkay.com'
               .hostname → 'www.darekkay.com'
               .port     → ''
               .pathname → '/projects/'
               .search   → '?filter=JS'
               .hash     → '#2'
               .href     → 'https://www.darekkay.com/projects/?filter=JS#2'
```
