## Settings

- Disable telemetry:
  - ☐ Options → Privacy & Security → Firefox Data Collection and Use
- Use logical tab order:
  - `browser.ctrlTab.recentlyUsedOrder`: `false`
- Customize startpage:
  - `about:preferences#home` → Firefox Home Content
- Disable Pocket:
  - `extensions.pocket.enabled: false`
